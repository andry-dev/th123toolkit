Controls

All controls available to the sprite editor are listed below:

Sprite selection scroll bar

Defines the current sprite to work on.

Group field

Defines the group number of the current sprite.

Index field

Defines the index number of the current sprite within its group.

X axis field

Defines the x axis of the current sprite.

Y axis field

Defines the y axis number of the current sprite.

Decompress on load

Defines if the sprite will be decompressed when loaded (default: disabled). Note: M.U.G.E.N. 1.+ only.

Show transparency color

If enabled, force the first color of the palette to fully opaque.

Switch to the sprite palette

If enabled, switch to the sprite palette automatically when changing the sprite selection scroll bar.

OnionSkin

If enabled, displays the OnionSkin sprite. When enabled, the OnionSkin scroll bar will be set to the current sprite. You can control the OnionSkin appearance in the Options window.

OnionSkin scrool bar

Defines the current OnionSkin sprite to display.

Workspace

The Sprite editor is used to manipulate the sprites (images) of the project. The graphic workspace contains the rulers, guide lines, OnionSkin image, current sprite image and the axis cross. Some of these can be enabled/disabled in the View menu.

You can change the position of the image by dragging it to the desirable position. The guide lines can be placed in the same way. All other points will move the axis and background image.

The zoom can be changed using the mouse wheel.
