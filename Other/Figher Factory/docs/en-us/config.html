Configuring

When you open Fighter Factory 3 at the first time, the default configuration is:

  • Classic interface (preset, icons, style and colors).
  • All confirmations and notifications active.
  • Balanced memory usage (memory cache and stream compression).
  • Font Courier 10, with the System encoding as default when UED can't do the detection.
  • No OpenGL hardware acceleration.
  • Left panel, right panel, rulers, state controller list and CLSNs visible.
  • English language.

Fighter Factory 3 keeps the configuration between sessions, including some editor states.

You can change the configuration in two places, described below.

Menu View

This menu defines the visibility and appearance of some controls. For more information see View...

Menu Tools / Options

The Options window of Fighter Factory 3 contains all customizable settings. For more information see Options...
