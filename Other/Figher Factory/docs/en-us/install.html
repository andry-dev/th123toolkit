Installation

Fighter Factory 3 doesn't require any special procedure to be installed. Only extract the files in any folder that doesn't require elevated privilegies.

File structure

  • Codecs
  • Colorsets: Contains all available sets of colors to be used in the Fighter Factory interface.
  • Docs: Contains de Reference Documentation.
  • Iconsets: Contains all available sets of icons to be used in the Fighter Factory interface.
  • Languages: Contains all available languages.
  • PalFX: Contains all available PalFX templates.
  • Presets: Contains all interface presets.
  • Scripts: Contains all Lua scripts.
  • SCtrls: Contains all state controller templates.
  • Snippets: Contains all available code snippets.
  • Styles: Contains all available interface style sheets.
  • FF3.exe: The program executable.
  • Completer.xml: The auto-completer data base file
  • Config.xml: The configuration data base file
  • History.xml: The file history data base file
  • Shortcuts.xml: The shortcuts data base file
  • Syntax_mugen.xml: The syntax highlight data base file for M.U.G.E.N.
  • Syntax_lua.xml: The syntax highlight data base file for Lua
  • libgcc_s_dw2-1.dll
  • mingwm10.dll
  • phonon4.dll
  • QtCore4.dll
  • QtGui4.dll
  • QtNetwork4.dll
  • QtOpenGL4.dll
  • QtWebKit4.dll
  • QtXml4.dll
