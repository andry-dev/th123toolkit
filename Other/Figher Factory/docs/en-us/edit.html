Menu Edit

The Edit menu is used to perform some common actions in the current project/editor. The options are described below.

Undo

Undo a change.

Redo

Redo a change.

Cut

Copy to the clipboard and remove the selected text/item. Note: Only on Definitions, Commands, States, Sprites and Animations.

Copy

Copy the selected text/item to the clipboard. Note: Only on Definitions, Commands, States, Sprites and Animations.

Paste

Insert the contents of the clipboard if are in a supported format. Note: Only on Definitions, Commands, States, Sprites and Animations.

Find / Replace

Enable the quick search mode that perform a search/replace operation in the current editor. You can enable some options to give more control over the search: case sensivity, backward search, circular search and regular expressions. Note: Animations are searched by number or name. Sprites and sounds are searched by 'Group,Index'.

Find / Replace (multi-line)

Perform a multi-line search/replace operation in the current editor. Note: Only on Definitions, Commands and States.

Code

Displays the Code menu.
