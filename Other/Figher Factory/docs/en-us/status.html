Status bar

The Status bar will contain additional informations.

Each editor displays its own information as described below:

Definitons: Current row, current oolumn, line count, encoding, syntax error count.

Sprites: Current cursor position (X, Y), horizontal and vertical guide line positions (axis-based), SFF version.

Animations: Current cursor position (X, Y), horizontal and vertical guide line positions (based on the sprite axis).

Commands: Current row, current oolumn, line count, encoding, syntax error count, unique command count.

States: Current row, current oolumn, line count, encoding, syntax error count.

Sounds: Nothing.

In all editors, the last section will contain the type of the current project and the version of M.U.G.E.N. assigned to it.

Note: The SFF version can be changed any time by clicking in the status bar.
