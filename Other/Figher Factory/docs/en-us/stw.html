Controls

All controls available to the states editor are listed below:

Text block list

Displays a tree list with all text blocks of the document. Some b.

New State controller

Creates a new State controller template.

Edit selected State controller

Edit the selected State controller template.

Delete selected State controller

Deletes the selected State controller template.

State controller list

Displays a list with all available State controller templates. To insert a template you must double-click an item in the list.

Workspace

As all other text based workspaces, the definitions editor have syntax highlight and checking, and respect all text and color options.
