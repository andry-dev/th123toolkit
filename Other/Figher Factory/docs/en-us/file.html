Menu File

The File menu is used to manipulate you project. You can create, open, save, test, among others. The options are described below.

New

Creates a new project. The project template window will be shown. This window displays all available template scripts.

Open

Open an existing project from a definition file. Fighter Factory 3 automatically detects if the project is a character, stage, font or storyboard, and loads all files properly. The project will be added to the history for use in the Open recent menu.

Open recent

Open a project from the history. If the file doesn't exists, will be removed from the list. If the selected project exists, will be moved to the top of the list.

Close

Closes the active project. If any file is currently modified and not been saved, asks for confirmation. If the project is the only one, Fighter Factory will add a new blank.

Close all

Closes all projects, and adds a new blank. If any file is currently modified and not been saved, asks for confirmation.

Close others

Closes all projects except the current. If any file is currently modified and not been saved, asks for confirmation.

Save

Save all files of the current project, overwriting the old one. If the file doesn't exists, execute Save as.

Save as

Displays a Save dialog for every file of the current project.

Create backup

Creates a backup of the current project. All files will be saved in the sub folder 'Backup/date-time', in the same directory of the definitions file. The date-time refer to the format day-month-year hour-minute-second ap. Every time you use this function, a new sub folder will be created.

Load backup

Displays a window with all backups stored in the 'Backup' folder of the current project. When you hit OK, the selected backup will be loaded. Note: This function don't overwrites the original files. To do it you need to hit 'Save'.

Run in M.U.G.E.N.

Displays a window with all versions of M.U.G.E.N. added in the Options window. Saves the current project in a temporary folder in the M.U.G.E.N. directory based on the selected item. Executes M.U.G.E.N. with the parameters specified on the selected item. Fighter Factory only will return the control to the user when M.U.G.E.N. close.

Exit

Closes Fighter Factory. This function will confirm the close action if it is defined in the Options window.
