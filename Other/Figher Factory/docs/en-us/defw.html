Controls

All controls available to the sprite editor are listed below:

Text block list

Displays a tree list with all text blocks of the document. Some blocks are handled in a special way to display more informations. If you click in the list, the cursor will be placed in the line of the selected item.

Workspace

As all other text based workspaces, the definitions editor have syntax highlight and checking, and respect all text and color options.
