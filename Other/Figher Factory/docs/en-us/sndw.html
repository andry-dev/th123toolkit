Controls

All controls available to the sprite editor are listed below:

Sound selection scroll bar

Defines the current sound to work on.

Group field

Defines the group number of the current sound.

Index field

Defines the index number of the current sound within its group.

Informations

Displays some informations about the current sound:

Channels: number of channels.

Sample rate: number of samples per second.

Duration: the duration in seconds.
