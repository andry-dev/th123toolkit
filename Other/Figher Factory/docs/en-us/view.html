Menu View

The View menu is used to define the visibility and appearance of some interface controls. The options are described below.

Left panel

Defines the visibility of the left panel.

Right panel

Defines the visibility of the right panel. Note: This panel doesn't exist in all editors.

Organizer

Defines the visibility of the Organizer in the right panel. Note: Only on Sprites, Animations, Palettes and Sounds.

Action list

Defines the visibility of the Action list window. This window displays all possible actions for the current editor. Click on an item in the list to execute the action.

Calculator

Defines the visibility of the Calculator.

Clipboard container

Defines the visibility of the Clipboard container. Within this, you can store named copies to be pasted later.

State controller list

Defines the visibility of the State Controller list. Note: Only on States.

Text toolbar

Defines the visibility of the Text toolbar. This toolbar contains the code snippet system and a syntax error navigation list.

Centralize axis

Centralize the axis on the graphic workspace.

Sprite axis

Defines the visibility of the sprite crosshair.

Animation axis

Defines the visibility of the animation crosshair.

Extra cross

Defines the visibility of the extra cross. Note: Only on Animations.

Rulers

Defines the visibility of the rulers in the graphic workspace.

Guide lines

Defines the visibility of the guide lines in the graphic workspace.

CLSN 1

Defines the visibility of the CLSN 1. Note: Only on Animations.

CLSN 2

Defines the visibility of the CLSN 2. Note: Only on Animations.

CLSN default

Defines the visibility of the CLSN 1 default. Note: Only on Animations.

CLSN default

Defines the visibility of the CLSN 2 default. Note: Only on Animations.

Zoom

Defines the zoom level of the graphic workspace. The options are self explanatory.
