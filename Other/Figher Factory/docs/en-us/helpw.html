Controls

The Reference documentation is handled as an editor, but isn't. You can exit this mode by switching to another editor.

This workspace is like an Web Browser that displays the Fighter Factory 3 documentation. Any external reference is blocked for security reasons.

The documentation is shown in the current language, if available. Otherwise the English version will be used.
