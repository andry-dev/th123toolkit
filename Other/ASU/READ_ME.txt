Version 10 - 31/08/2013

	Added unpacking of multiple images
	Image palettes / colour tables / indexes should now be preserved
	Better error messages
	Performance improvements
	Minor UI tweaks
============================================================
Version 9 - 10/08/2012

	Added option to not use the choose folder dialog box 
	Added option to not open destination folder automatically
	Removed all .NET 3+ dependancies, now runs on just .NET 2 to improve compatibility
	Minor UI tweaks
============================================================
Version 8 - 27/11/2011

	Added select all functionality
============================================================
Version 7 - 06/10/2011

	Fixed bug when exporting frames when Options form has not been loaded
============================================================
Version 6 - 19/09/2011

	Added integration with NConvert command line image file conversion utility to allow ASU to support 400+ file formats, both imported sprite sheets and exported frames
	Added automatic border, hover and selection colours to contrast with sprite sheet
	Major Options form layout changes
============================================================
Version 5 - 04/09/2011

	Added option to change exported file format
	Fixed out of memory exception when unpacking very large spritesheets
	Improved unpacking percentage calculation between 10 - 20 %
	Minor UI tweaks
============================================================
Version 4 - 31/08/2011

	Added 'Cut' functionality to allow frames to be cut into seperate frames
	Improved magnified area bitmap quality
	Added control of mouse cursor via keyboard arrow keys to allow pixel perfect cutting/ selecting
	Minor UI tweaks
============================================================
Version 3 - 19/08/2011

	Fixed bug when Reload is clicked before image is loaded
	Fixed tab order on all forms

	Fixed unpacking percentage calculation when pc under 10%
