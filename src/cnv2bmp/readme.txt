東方緋想天 イメージファイル変換ツール ver1.01

■使い方
  変換したいファイルのある場所にcnv2bmp.exeを置いて実行するだけです。
  ただし、キャラクターパターン画像を2Pカラーで出力したい場合は、以下のように指定してください。

  例）cnv2bmp -2p

■お約束
  ツールの質問やツールで展開したファイルの質問を本家にしたりしないように！

  東方緋想天の著作権は「黄昏フロンティア」「上海アリス幻樂団」に帰属します。
  利用は個人の使用範囲内でお願いします。

■開発環境
  WindowsXP SP2
  Visual C++ Toolkit 2003 ＋ Platform SDK ＋ Visual Studio .NET 2003

■対応ファイル
  ・8bit、24bit、32bitイメージファイル（*.cnv）
